# GRAM-TALK
그램의 라이트닝 토크입니다



## 규칙

- 전날에 주제를 선정해 슬랙 `gram-talk`채널에 올립니다.
- 시각자료는 발표당일까지 깃허브에 업로드합니다 
  - yyyymmdd 경로에 업로드합니다.
  - 파일, 영상 네이밍: yyyymmdd-000
    - 예시 - 20191113-김다은
  - 커밋 메시지: 커밋내용 파일명
- 당일 시각자료 노션 업로드
- 발표 시 존댓말을 사용합니다.
- 발표 시작할 때에는 안녕하세요 ~입니다라는 인삿말 후 시작합니다.
- 발표를 마칠 때에는 감사합니다라는 인삿말 후 마칩니다.
- 질문은 마지막에 Q/A시간에 합니다.
- 수요일 8교시에 진행됩니다.
- 다음시간에 그램톡 회고를 별도로 진행하여 서로 이야기를 나눕니다.
